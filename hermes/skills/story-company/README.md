# Story Company Hermes Skills

This directory stores learner-owned Hermes skill extensions for Story company workflows.

Do not duplicate skills already bundled by Hermes, such as the native diagramming skill. Story-specific extensions should live here only when they add durable company process that is not already covered by the Hermes library.

Skills in this pack are copied into `$HERMES_HOME/skills/story-company` by the Hermes learner. Local PVC edits are preserved until an eval-passing promotion branch updates the Git-canonical copy.
