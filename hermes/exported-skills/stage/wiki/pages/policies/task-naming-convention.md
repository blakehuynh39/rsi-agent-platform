# Task Naming Convention

## Summary

Convention for naming tasks in the EU Launch project: prefix with role identifier, then story name, then task description.

## Claims

- Task names must start with a team or role identifier: [design] for UX/UI, [fe] for frontend, [be] for backend (SDK/API), [eng] for tasks spanning both frontend and backend. `claim:claim_2_1` `confidence:1.00`
