# AA Wallet Infrastructure Setup

## Summary

Guide for setting up an Account Abstraction (AA) wallet using ZeroDev SDK, Pimlico bundler/paymaster, and Viem on Sepolia testnet. Covers wallet initialization with master and regular signers, session key generation, and session key revocation.
