# Numo

## Summary

Numo is a crypto project that processes user submissions, validates them, and pays rewards. It uses TaxBandits for W-8/W-9 tax forms and maintains daily statistics on submissions and validated/unvalidated payouts. The team avoids explicitly naming Numo in contexts where customers might dislike crypto.
