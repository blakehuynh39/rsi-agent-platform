# Web2 Partner Integration Solution

## Summary

A near-term solution to help web2 partners integrate with the Story protocol by abstracting away web3 complexities such as wallet creation and transaction management. It provides a web2 API for IP registration, licensing, and remixing, with options for gas sponsorship.
