# Eng Leads Offsite April 2025

## Summary

Engineering leads offsite held in Vancouver, Canada from April 23-26, 2025. The offsite aimed to define the architecture for Project Poseidon, discuss TGE scope, define a Proof of Concept, and plan for an August Alpha release. Thirteen attendees were expected, with remote participation from three individuals. Goals included architecture definition, resource allocation, and deliverables like architecture diagrams, product design documents, and API designs.
