# Platform Services Initiative

## Summary

Initiative to build Platform Services that bridge the gap between the core protocol SDK and the IP ecosystem needs for Emergence Universe (EU). It includes utilities like content uploading, NLP post processing, reading experience, user profiles, wallet management, and analytics, with a core value proposition of indexing and preparing data for franchises.
