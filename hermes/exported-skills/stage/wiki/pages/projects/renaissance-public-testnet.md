# Renaissance Public Testnet

## Summary

The Renaissance Public Testnet is scheduled for release on April 15. Its goals include integrating with infrastructure providers, gathering feedback on infrastructure and developer experience, marking the protocol code freeze milestone, and enabling external builders to test on Renaissance. Key deliverables span an information page, onboarding page, documentation, rollup deployment, platform SDK/API release, applications, and launch partners.
