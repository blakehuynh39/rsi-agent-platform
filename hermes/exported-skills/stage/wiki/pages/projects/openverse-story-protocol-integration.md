# Openverse ↔ Story Protocol Integration (57B)

## Summary

Project to register openly licensed and public domain works from Openverse (images and audio) as IP Assets on Story Protocol Beta, using a curated set of 100 top keywords/topics with 100 assets each, totaling 10,000 assets.
