# Aeneid Portal (aeneid.portal.story.foundation) Domain Configuration

## Summary

Configuration, SSL issue, and resolution for the aeneid.portal.story.foundation domain, which is a portal subdomain hosted on Vercel and proxied via Cloudflare. An SSL cipher mismatch occurred due to a Cloudflare limitation with multi-level subdomain wildcard certificates; the immediate fix was to disable Cloudflare proxying, with a planned migration to a single-level subdomain.
