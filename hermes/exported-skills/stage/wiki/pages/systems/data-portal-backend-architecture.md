# Data Portal Backend Architecture

## Summary

Current architecture for the Data Portal Backend as of May 19, 2026. Story ingests provider payloads into a standardized Trace Schema v1.0, stores durable traces in DynamoDB with explicit search indexes, and exposes read APIs. The system uses an async ingestion pipeline with SQS and a data-audit-ingestor worker. On-chain commitment is limited to hash-only anchors.
