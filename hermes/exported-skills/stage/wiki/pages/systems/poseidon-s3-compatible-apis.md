# Poseidon S3 Compatible APIs

## Summary

Poseidon provides an S3-compatible API supporting a subset of AWS S3 operations for bucket management, object operations, and multipart uploads.

## Claims

- Poseidon supports the following S3 API operations: CreateBucket, ListBuckets, ListObjectsV2, HeadObject, GetObject, CreateMultipartUpload, CompleteMultipartUpload, ListParts. `claim:claim_1_1` `confidence:1.00`
