# PIP-Labs-RE GitHub Organization and Docs Repository

## Summary

The PIP-Labs-RE GitHub organization was created to host the docs repository after enterprise-level branch protection rules prevented direct pushes to main, as desired by the sole maintainer Jacob. The repository is public with lenient rules. Collaborators were invited to the organization and repo. There is consideration to require at least one approver due to the public-facing nature of the docs.
