# IPAccount Design

## Summary

IPAccount is not a generic AA wallet; it is customized for Programmable IP use cases and does not support owner calls to arbitrary external contracts.

## Claims

- IPAccount does not serve as a generic AA wallet; it is customized for Programmable IP, and does not support the owner calling any external contracts. `claim:claim_2_1` `confidence:1.00`
