# Trace Backend Architecture

## Summary

Staging V1 of the Trace backend is deployed and verified as of June 3, 2026. It provides a pipeline for providers to submit data audit records, idempotent writes, exact-match search indexes, and scoped grouping. The system runs on AWS with SQS, DynamoDB, and S3, and is accessed via story-api.
