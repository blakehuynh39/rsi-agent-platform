# Royalty Graph

## Summary

Royalty Graph processes royalty calculations and transactions, with v1 and v2 versions. Usage data is primarily accessed via Temporal Cloud namespace; internal dashboards for transaction/royalty metrics are unconfirmed. Monitoring via Grafana and Prometheus is currently absent for royalty-specific metrics.
