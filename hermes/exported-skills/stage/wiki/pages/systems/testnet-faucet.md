# Testnet Faucet (Aeneid)

## Summary

The Aeneid testnet faucet distributes IP tokens to developer wallets. The frontend is at aeneid.faucet.story.foundation, which calls the backend at 35.207.0.103:23313 (GCP instance use1-aeneid-fuacet). The backend validates requests, records wallet addresses in a DB, and batch-sends tokens. Development by S07ASS1JQNP. Migration to AWS is in progress.
