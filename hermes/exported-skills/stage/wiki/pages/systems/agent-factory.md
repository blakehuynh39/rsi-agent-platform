# Agent Factory

## Summary

Agent Factory is an internal agent management platform created by Jinn. It initially hosted personal agents and is migrating to functional agents. It currently runs agents such as aiwei-agent and blockchain-agent, and recently adopted LiteLLM for inference routing.
