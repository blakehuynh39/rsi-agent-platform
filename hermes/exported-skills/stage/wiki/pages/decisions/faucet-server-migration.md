# Faucet Server Migration from GCP to AWS

## Summary

The faucet server use1-aeneid-fuacet on GCP (IP 35.207.0.103) is being migrated to AWS. It currently serves https://aeneid.faucet.story.foundation/ with a backend that validates requests and dispatches testnet tokens. A domain-based frontend is planned to simplify the migration.
