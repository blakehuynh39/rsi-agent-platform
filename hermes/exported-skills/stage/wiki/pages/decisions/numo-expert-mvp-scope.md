# MVP Implementation Scope

## Summary

MVP must-have features: task list/detail, submit/draft/skip, audio player, transcript editor, manual/team assignment, profile gating, basic quiz, honeypot metadata, consensus config, admin submission view, duration tracking. Should-have: reviewer queue, expert judge team, quality score, shadow mode, quickstart docs, FAQ/walkthrough. Later: automatic routing, full payout engine, mobile HITL, deepfake detection, credential-based expert tasks.
