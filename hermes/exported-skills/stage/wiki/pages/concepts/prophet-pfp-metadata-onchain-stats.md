# Prophet PFP Metadata to Reflect On-Chain Protocol Stats

## Summary

The concept involves NFT metadata dynamically reflecting on-chain statistics such as number of collects, relationships, license type. It aims to provide a fun and convenient showcase of IP Asset stats. Still in idea stage, with technical feasibility under assessment. Proposed engineering approach uses IPFS/Arweave for metadata hosting with a dynamic API endpoint.
