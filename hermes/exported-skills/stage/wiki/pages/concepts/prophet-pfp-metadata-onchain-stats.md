# Prophet PFP Metadata to Reflect On-Chain Protocol Stats

## Summary

The concept involves NFT metadata dynamically reflecting on-chain statistics such as number of collects, relationships, license type. It aims to provide a fun and convenient showcase of IP Asset statistics, but is still in the idea stage and requires technical feasibility assessment. One proposed engineering approach uses IPFS/Arweave with a dynamic API endpoint.
