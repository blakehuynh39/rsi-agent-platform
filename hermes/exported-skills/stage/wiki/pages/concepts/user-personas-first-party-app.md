# User Personas for First Party App

## Summary

Describes the target user personas for the First Party App: IPs and Projects (e.g., Azuki, Bored Ape, Clone X) with goals to delegate IP distribution, foster collaboration, establish a robust ecosystem, and enhance user experience.
