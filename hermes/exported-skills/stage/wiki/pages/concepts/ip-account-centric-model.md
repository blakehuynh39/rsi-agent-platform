# IP Account Centric Model

## Summary

The IP Account Centric Model conceptualizes IPAccount as the core identity (noun) and Modules as actions (verbs). IP Accounts represent IP as real entities on Ethereum, with IP NFTs serving as identity tokens. IPAccounts can hold license/royalty NFTs, store state, and initiate actions via modules, while an Orchestrator authorizes interactions.
