# Shared Ownership of IP Assets in Magma

## Summary

Discussion about enabling multiple authors to share ownership of an IPA in Magma, including control over permissions and royalties. Currently, only the minter/owner can control these. The team is considering using the permission system to support shared ownership, but clarification is needed.
