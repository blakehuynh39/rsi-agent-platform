# ElevenLabs Voice Agent Orchestration Paths

## Summary

Overview of three approaches to building voice agents with ElevenLabs: Fully Managed, Managed + Custom LLM Server, DIY. Covers tradeoffs in pipeline control, session state, RAG, infrastructure, latency, and when to choose each path.
