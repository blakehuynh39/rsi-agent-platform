# PIL Derivative commercialRevShare Override

## Summary

Proposal to allow derivative IP owners to override the commercialRevShare field in the PIL license when registering a derivative, addressing a use case from Mahojin.

## Claims

- Mahojin raised a use case where a derivative IP owner wants to change the commercialRevShare inherited from the parent license (e.g., parent sets 5%, child wants 15%). `claim:claim_1_1` `confidence:1.00`
